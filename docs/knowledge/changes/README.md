# Registro de cambios de conocimiento

Cada tarea cerrada agrega aquí un archivo Markdown basado en `.agents/templates/knowledge-update.md`.

Convención: `AAAA-MM-DD-slug-del-cambio.md`.

Este registro es el historial detallado; `../knowledge-tree.html` es el mapa navegable y `../knowledge-graph.html` es el mapa de relaciones. Leer primero los registros vinculados al nodo de trabajo, no todo el historial.
